# TokoOnlineMengunakanGolang
Basic E-commerce Web App built with GO

## Features
- Home Page
- Register, Login, Profile User
- Products Page
- Single Product Page
- Shopping Cart
- Checkout
  - Shipping Cost Calculation
  - Payment Gateway
- Order History
- Admin Panel:
  - Dashboard
  - Manage Products
  - Manage Orders
  - Manage Users
  - Reports
  - Configuration
